variable HD_WriteAsyncResponseProto(string cmdID, string returnString)¶

Prototype function to allow HDF5 function to write async responses, with return strings, to the WSE.

static variable HD_WriteAsyncResponseWrapper(string cmdID, string returnString)¶

Wrapper for the optional tango related function HD_WriteAsyncResponseWrapper The approach here using a function reference and an interpreted string like $"" allows to convert the dependency on the function TI_WriteAsyncResponse from compile time to runtime. This function will call TI_WriteAsyncResponse if it can be found, otherwise HD_WriteAsyncResponseProto is called.

variable HD_WriteAckProto(string cmdID, variable returnValue)¶

static variable HD_WriteAckWrapper(string cmdID, variable returnValue)¶

Wrapper for the optional tango related function HD_WriteAckWrapper The approach here using a function reference and an interpreted string like $"" allows to convert the dependency on the function TI_WriteAck from compile time to runtime. This function will call TI_WriteAck if it can be found, otherwise HD_WriteAckProto is called.
