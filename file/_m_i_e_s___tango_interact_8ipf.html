run the baseline QC check from the WSE. This will zero the amp using the pipette offset function call, and look at the baselineSSAvg, already calculated during the TestPulse.

The EXTPINBATH wave will also be run as a way of making sure the baseline is recorded into the data set for post-experiment analysis
